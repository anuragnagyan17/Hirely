# 🚀 Job Portal Web Application

A full-stack **Job Portal Platform** that enables users to explore job opportunities, apply seamlessly, and allows employers to manage job postings efficiently.

---

## ✨ Features

* 🔐 Secure Authentication (JWT-based login/signup)
* 👨‍💼 Role-based Access (Candidate & Employer)
* 📝 Create, Update & Delete Job Listings
* 🔍 Search & Filter Jobs
* 📤 Apply for Jobs with Profile/Resume
* 📊 Dashboard for Users & Employers
* ☁️ Cloudinary Integration for File Uploads

---

## 🛠 Tech Stack

### Frontend

* React.js (Vite)
* React Router

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)

---

## 📂 Project Structure

backend/ → REST API (Node + Express)
frontend/ → UI (React + Vite)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

git clone https://github.com/your-username/your-repo-name.git

---

### 2️⃣ Backend Setup

cd backend
npm install
npm run dev

---

### 3️⃣ Frontend Setup

cd frontend
npm install
npm run dev

---

## 🔑 Environment Variables

Create a `.env` file in backend folder:

PORT=8000
MONGO_URI=your_mongodb_uri
SECRET_KEY=your_secret_key
